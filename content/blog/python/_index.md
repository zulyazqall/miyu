Python is a programming language that lets you work quickly and integrate systems more effectively. one of most popular Programming language, created by Guido van Rossum and released in 1991. Python become the first choice to start with programming language. It has a simple syntax and readability is the best choice for kids to understand programming concepts.

## Python Syntax

Python syntax refers to the **set of rules and conventions** and written as a sequence of statements. Each statement represents an action or a command. For example, assigning a value to a variable, calling a function, or defining a loop are all examples of statements.

## First Program

Mandatory for beginners in any programming language to start with "Hello World!" program.

```python
print("Hello World!")
```
**Output**
```Hello World!```
