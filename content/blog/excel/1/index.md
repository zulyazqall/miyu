A **Formula** is matematatical expresion used to calculate the value of the cell 


## Formulas

1. '+' addition
2. '-' substraction
3. '/' devision
4. '*' multiplication
5. '=' equal


## Structure
The structure of spreadsheet formula is ..
```
= [cell][operator][cell]
= A2 + A3
```

==REMEMBER==
formula ALWAYS start with `=`
